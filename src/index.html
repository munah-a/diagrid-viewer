JSON Format

The file must have nodes and beams arrays:

{
  "nodes": [
    { "id": 0, "x": 0.0, "y": 0.0, "z": 0.0 },
    { "id": 1, "x": 5.0, "y": 0.0, "z": 3.0 }
  ],
  "beams": [
    { "id": 0, "node_start": 0, "node_end": 1 }
  ]
}
Rules:
• Node id must be unique integers (0-based preferred)
• Coordinates x, y, z in metres
• Beam node_start / node_end must reference valid node ids
• Beam id must be unique integers
• No duplicate beams (same start–end pair)
Grasshopper Export Script

Use this GhPython component to export lines as a JSON model:

"""Grasshopper Python – export lines to diagrid JSON
Inputs:
  lines   - list of Line/Curve geometry
  path    - file path string (e.g. "C:\\model.json")
  export  - boolean toggle to trigger export
"""
import json
import Rhino.Geometry as rg

if export and lines and path:
    pts = {}
    nid = 0
    beams_out = []
    bid = 0

    def pt_key(p):
        return (round(p.X,6), round(p.Y,6), round(p.Z,6))

    def get_or_add(p):
        nonlocal nid
        k = pt_key(p)
        if k not in pts:
            pts[k] = {"id": nid, "x": p.X, "y": p.Y, "z": p.Z}
            nid += 1
        return pts[k]["id"]

    for ln in lines:
        if hasattr(ln, "PointAtStart"):
            p1, p2 = ln.PointAtStart, ln.PointAtEnd
        else:
            p1, p2 = rg.Point3d(ln.From), rg.Point3d(ln.To)
        n1 = get_or_add(p1)
        n2 = get_or_add(p2)
        if n1 != n2:
            beams_out.append({"id": bid, "node_start": n1,
                              "node_end": n2})
            bid += 1

    model = {
        "nodes": sorted(pts.values(), key=lambda n: n["id"]),
        "beams": beams_out
    }
    with open(path, "w") as f:
        json.dump(model, f, indent=2)
    result = "Exported %d nodes, %d beams" % (len(pts), len(beams_out))
else:
    result = "Set export=True, connect lines & path"
